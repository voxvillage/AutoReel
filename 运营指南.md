## 二、日常运营流程

### 每日（10分钟）

1. **网站健康检查** — 打开 https://voxvillage.github.io/AutoReel/，确认页面正常加载
2. **AI生成测试** — 用免费额度试一次脚本生成，确认 DeepSeek API 正常
3. **社媒内容发布** — 发布1-2条产品推广内容（见「社媒运营」章节）
4. **邮件检查** — 查看 qiuzhixu@icloud.com 是否有用户反馈
5. **数据记录** — 记录当日网站访问量、生成次数、付费咨询

### 每周（30分钟）

1. **GitHub Pages 检查** — 确认部署无错误
2. **PayPal 账户检查** — 查看是否有新交易
3. **竞品扫描** — 检查 AutoShorts.ai、Faceless.video 是否有新功能/价格变化
4. **内容复盘** — 分析本周社媒帖子的互动数据，优化下周内容
5. **趋势更新** — 在趋势雷达工具中尝试新 niche，发现内容灵感

### 每月（2小时）

1. **版本更新** — 检查 i18n 翻译、SEO meta 标签是否过时
2. **定价策略复盘** — 根据付费转化率调整定价文案
3. **功能迭代** — 从用户反馈中提炼新功能需求，排入路线图
4. **推广渠道评估** — 哪些渠道效果好？哪些要砍掉？
5. **收入分析** — 计算当月收入，对比预估，调整下月目标
6. **代码备份** — 通过 GitHub API 推送最新版本

## 五、故障应急

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 网站无法访问 | GitHub Pages构建失败 | 用API重新推送index.html |
| AI生成报错 | DeepSeek API余额不足 | 充值，或检查429限速 |
| PayPal SDK加载失败 | CDN故障 | 自动显示备用链接 |
| 付款后未激活 | localStorage被清除 | 刷新页面或清除缓存 |
| 用户投诉免费次数被多扣 | 8秒冷却连续点击 | 解释冷却机制 |
| 合规盾/原创/可信工具报错 | DeepSeek API限制 | 与其他工具共用同一key |
